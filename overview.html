Documentación del Juego "Tres en Raya"

Este proyecto implementa el juego "Tres en Raya" como parte del curso de Metodología de la Programación (2024-2025).

Requisitos

Para ejecutar este proyecto es necesario tener Java instalado (versión 11 o superior).

Paquetes del Proyecto

  • juego.control: Controla las reglas y el flujo del juego.
  • juego.modelo: Representa los componentes del juego (tablero, piezas, jugadores, etc.).
  • juego.textui: Proporciona la interfaz de usuario en modo texto.
  • juego.util: Contiene clases auxiliares como coordenadas y direcciones.
